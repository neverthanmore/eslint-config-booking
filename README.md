# eslint-config-booking

Node Style Guide for Booking.

## Install

```bash
npm i eslint eslint-config-booking --save-dev
```

## Usage

- `package.json`

```json
{
  "devDependencies": {
    "eslint-config-booking": "1",
    "eslint": "3"
  }
}
```

- `.eslintrc.js`

```js
module.exports = {
  extends: 'eslint-config-booking',
};
